# Predictive Analysis of Online Shoppers Purchasing Intent

This repository contains an end-to-end Machine Learning classification pipeline designed to analyze consumer behavior data and predict e-commerce purchase conversions.

## 🔍 Highlights
* Avoided data leakage by embedding scaling within Scikit-Learn `Pipelines`.
* Evaluated multiple models including Random Forest, Gradient Boosting, and XGBoost.
* Leveraged `StratifiedKFold` cross-validation to handle class imbalance.
* Performed hyperparameter optimization via `RandomizedSearchCV`.
